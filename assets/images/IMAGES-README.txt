Place all website images in this folder.

Recommended naming:
- hero-community.jpg       ← Main homepage photo
- mass-sunday.jpg          ← Mass photo
- community-gathering.jpg  ← Events/activities photo
- logo.png                 ← CCFC logo (if created)

Recommended image sizes:
- Hero/banner images: 1920×1080px
- Card/section images: 800×600px
- Keep file sizes under 500KB for fast loading (use tinypng.com to compress)
